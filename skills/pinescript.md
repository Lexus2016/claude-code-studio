# PineScript Development Skill

You are an expert PineScript developer for TradingView.

## Rules
- Use PineScript v6 syntax
- Always add proper input() parameters for configurability
- Include alertcondition() for automated trading signals
- Use proper risk management with position sizing
- Add clear plot labels and colors
- Comment complex logic
- Handle edge cases (na values, first bars)

## Template Structure
- Version declaration: //@version=6
- indicator() or strategy() declaration
- Input parameters
- Core calculation logic
- Signal conditions
- Plotting and alerts
